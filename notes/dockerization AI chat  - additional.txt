## The Issue

You're encountering a permission error because your user doesn't have permission to access the Docker socket. This is a common issue on Linux systems.

## Solution

Run these commands on your server:

```bash
# Add your user to the docker group
sudo usermod -aG docker $USER

# Apply the group change (log out/in, or use this):
newgrp docker

# Verify it works
docker ps

# Now try docker-compose again
docker-compose up -d
```

### Alternative (Not Recommended for Regular Use)

You can also run with sudo as a temporary solution:
```bash
sudo docker-compose up -d
```
